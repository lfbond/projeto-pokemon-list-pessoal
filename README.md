# Projeto Pokemon List - Semana Dev em Dobro

Esse projeto fizemos uma lista, estilo Pokedex com 10 pokemons, usando HTML, CSS e JavaScript puro.
Mas essa versão foi a minha 1ª versão, acrescentando todos os Pokemons da 1ª Temporada (Kanto - 151 pokemons), 
e modificando alguns detalhes que achei mais legal colocar e alterar do original.

Farei novas melhorias futuramente, com mais temporadas.
